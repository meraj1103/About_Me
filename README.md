## ✨ About This Project

This is my personal **GitHub portfolio page** — a single-page application crafted entirely with **vanilla HTML, CSS, and JavaScript** (no frameworks, no dependencies). It was designed to make a strong first impression on recruiters, collaborators, and fellow developers visiting my GitHub profile.

The design philosophy centers on a **dark, professional aesthetic** with vibrant indigo/cyan/purple accents, smooth micro-animations, and an interactive particle canvas that reacts to mouse movement — giving the site a premium, modern feel that stands out.

---

## 🎯 Features

- 🌌 **Animated Particle Hero** — 90 interactive canvas particles that respond to mouse movement with connection lines and repulsion physics
- ⌨️ **Typewriter Effect** — Cycles through phrases like *"scalable software"*, *"beautiful UIs"*, *"impactful products"*
- 📊 **Animated Skill Bars** — Progress bars that fill smoothly when scrolled into view
- 🃏 **3D Tilt Cards** — Skill and project cards tilt in 3D on mouse hover
- 🎞️ **Scroll Reveal Animations** — Every section fades and slides in as you scroll
- 📱 **Fully Responsive** — Looks great on desktop, tablet, and mobile
- 📬 **Working Contact Form** — With animated success state (plug in your backend/EmailJS)
- 🔗 **Real Project Links** — Live demos for CampusConnect and Sahaya (hosted on Firebase)

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| **HTML5** | Semantic page structure |

| **Vanilla JavaScript** | Particle engine, typewriter, scroll reveal, tilt, counters |

## 📦 Getting Started

No installation needed. Just clone and open:

```bash
git clone https://github.com/meraj1103/meraj1103.github.io.git
cd meraj1103.github.io
# Open index.html in your browser
```

Or run a local server:

```bash
python -m http.server 8080
# Visit http://localhost:8080
```
